# Malmmoderns runring v1

Built-in image generation used the production dungeon loot sheet as its visual reference.

> Create one isolated legendary runic ring inventory sprite matching the dark painterly late-1990s action-RPG loot style. Forge it from ancient black iron and tarnished Swedish silver, with a broad oval bezel, a subtle three-crowns motif, an emerald ore rune and thin blood-red mineral veins. Use a readable three-quarter silhouette on a perfectly flat `#FF00FF` chroma-key background. No hand, text, box, shadow or watermark.

The production alpha PNG is downsampled into a dedicated 22-pixel unique-loot icon by the asset builder.
