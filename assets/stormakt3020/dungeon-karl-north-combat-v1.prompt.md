# Karl north-facing combat sprites

Generated with the built-in imagegen workflow using `dungeon-karl-combat-v1.png` as the strict character and style reference.

The four equal cells show Karl XII from behind: sword windup, contact slash, follow-through, and guard/parry. The production PNG removes only the connected black background; the asset builder then uses fixed source-cell scaling so weapon arcs cannot change Karl's apparent body size.
