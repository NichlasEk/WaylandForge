# Temple seal fog v1

Built-in image generation was given the in-game screenshot as a style and scale reference.

> Create a clean 2x2 sprite sheet containing four animation frames of one narrow vertical enchanted-fog barrier. Use wispy charcoal smoke, a restrained emerald inner glow, deep blood-red veins and sparse ancient rune sparks. Match a late-1990s dark Nordic action-RPG pixel-art aesthetic. Each plume must be tall, narrow, isolated and suitable for vertical overlap. Use a perfectly flat `#FF00FF` chroma-key background. No UI, gates, characters, crosses, yellow, cyan lightning, text or watermark.

The production PNG uses local soft-matte chroma removal; runtime overlaps and alternates the four frames along every locked temple boundary.
