# Stormakt bridge, cannon and projectile assets v1

Generated with the built-in image generation model on 2026-07-11. The active Stora Bält environment and Danish boss sheets were supplied as style references only.

Prompt summary:

- Strict 3x3 production sheet on uniform `#00ff00` chroma key.
- Row 1: snapped bridge deck slab, twisted rail/truss with severed cables, shattered bridge machinery/gear chunk.
- Row 2: intact top-down Danish bridge cannon, detachable Kronens Tiende broadside cannon, destroyed matching bridge cannon.
- Row 3: fast red aimed shell, royal white broadside spear and circular golden tax-seal orb.
- Crisp late-arcade/Neo Geo 16-bit pixel art matching the existing dark iron/copper environment and red-white/brass Danish fleet.
- No text, borders, UI, characters, ships, smoke clouds, cast shadows or long motion trails.

The full generation request specified identical upper-left lighting, isolated cells, strong silhouettes under aggressive downscaling and projectile colors distinct from player fire and the nebula. The chroma-key original is `stormakt-bridge-cannons-projectiles-v1-source.png`; the runtime alpha sheet is `stormakt-bridge-cannons-projectiles-v1.png`, produced with border auto-key, soft matte, despill, transparent threshold 12 and opaque threshold 220.
