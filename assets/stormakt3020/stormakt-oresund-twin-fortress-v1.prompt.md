# stormakt-oresund-twin-fortress-v1

Generation mode: built-in image generation followed by the imagegen skill's deterministic chroma-key cleanup.

Prompt:

> Use case: stylized-concept. Asset type: production boss sprite sheet for the retro 1990s dark Scandinavian science-fantasy top-down shoot-em-up Stormakt 3020. Create exactly four large isolated fortress subjects in a clean 2x2 grid on a perfectly flat solid #ff00ff chroma-key background. Strict top-down / slightly elevated orthographic game view, coherent scale and lighting, generous uniform magenta gutters, no overlap, no cast shadows, no floor, no text, no UI, no ships, no people. Top-left: HELSINGÖR, the left Danish orbital bridge fortress half, broad black-gunmetal coastal bastion with red-white Dannebrog armor panels, brass cannon ports, gothic spires and a bright cyan crown-power node on its inward-facing right edge. Top-right: HELSINGBORG, the right Swedish-facing captured bridge fortress half, complementary mirrored architecture with worn Swedish-blue patina mixed with Danish red-white occupation plates, brass shield machinery, gothic spires and a bright cyan crown-power node on its inward-facing left edge. Bottom-left: the same Helsingör half heavily damaged but structurally recognizable, cracked armor, dark smoke vents represented as solid machinery rather than transparent smoke, orange embers, exposed cyan cables. Bottom-right: the same Helsingborg half heavily damaged but structurally recognizable, cracked armor, orange embers, exposed cyan cables. Both intact halves must read as separate inward-facing anchors of one enormous bridge gate, not spaceships. Gritty painterly pre-rendered pixel-art texture, 1990s PC game aesthetic, cold black iron, dirty blue patina, Danish signal red, warm brass, readable when reduced to about 90-120 pixels. The #ff00ff background must remain perfectly uniform with no gradients, texture, reflection or lighting variation, and #ff00ff must not appear in the subjects.

The untouched generation is preserved as `stormakt-oresund-twin-fortress-v1-source.png`. The production PNG was auto-keyed from the border with a soft matte, transparent threshold 12, opaque threshold 220, despill and forced alpha. The packer crops the four quadrants independently, retains only each cell's largest alpha component and scales every fortress into a 132x112 maximum box.
